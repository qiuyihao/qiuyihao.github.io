Install freeradius on CentOS 7.2

Posted on 2017-11-27

Step 1: install freeRadius

yum install freeradius

Step 2: users configuration

vi /etc/raddb/users

guest   Cleartext-Password := "guest-password"
    Service-Type = Framed-User,
    Framed-Protocol = PPP,
    Framed-IP-Address = 172.16.3.33,
    Framed-IP-Netmask = 255.255.255.0,
    Framed-Routing = Broadcast-Listen,
    Framed-Filter-Id = "std.ppp",
    Framed-MTU = 1500,
    Framed-Compression = Van-Jacobsen-TCP-IP

Step 3: clients configuration

vi /etc/raddb/clients.conf

client clients_all {
        ipaddr          = 0.0.0.0
        secret          = 487dc4f4cf94bc9427fa441d86b82f53
}

Step 4: test
