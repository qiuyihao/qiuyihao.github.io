Deploy portal router and wireless access point device

Veröffentlicht am 2017-11-23

Core Router (CR)：TP-LINK TL-ER5110G
Access Point (AP)：TP-LINK TL-WVR308 (无线VPN路由器代替)

CR配置

接口名称 IP地址 子网掩码 网关地址 首选DNS服务器
WAN 192.168.1.177 255.255.255.0 192.168.1.1 192.168.1.1
LAN 192.168.88.1 255.255.255.0 – –

AP配置

接口名称 IP地址 子网掩码 网关地址 首选DNS服务器
WAN 192.168.1.89 255.255.255.0 192.168.1.1 192.168.1.1
LAN 192.168.89.1 255.255.255.0 – – 
标注：
  1. CR WAN接入工位网线，LAN1接入AP的LAN8 (切勿接入AP的WAN)，LAN2接入台式机；
  2. AP关闭DHCP功能；
