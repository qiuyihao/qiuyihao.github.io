How to deploy a wifidog authorize server

Posted on 2017-11-26

实现流程：

第一步：客户端连接网络，访问任意HTTP外网网页，客户端连接上网络后，打开浏览器访问任意HTTP外网网页，触发Portal认证。

第二步：路由器拦截客户端访问外网的GET数据包，并重定向到WEB服务器。

没有通过认证的客户端发往外网的GET数据包会被企业路由器拦截，并且会向客户端返回一条重定向条目（假设WEB服务器域名为www.abc.com）, 重定向条目为http://www.abc.com/?gw_address=xxx&gw_port=xxx&gw_id=xxx&ip=xxx&mac=xxx&url=xxx。

注意：若WEB服务器在路由器外网，URL地址需要在路由器设置免认证策略，若WEB服务器端口为非80端口，还需要将WEB服务器的端口设置免认证策略。

第三步：客户端访问WEB服务器。客户端根据第二步返回的重定向条目与WEB服务器建立连接。

第四步：WEB服务器向客户端返回认证页面。WEB服务器向无线终端返回认证登录页面，针对该认证登录页面，需满足以下规范：

[1] 认证页面必须有一个Form的action= http:// LAN_IP: Port/wifidog/logincheck/，其中LAN_IP为当前企业路由器 LAN口的IP地址，Port为Portal服务端口，默认为8080；

[2] 认证登录页面以Get方式提交Form表单；

[3] 认证登录页面必须包含以下参数：

>
| 参数 | 说明 |
| — | — |
| user | 用户名 |
| pwd | 密码 |
| ip | 认证终端的IP地址 |
| mac | 认证终端的MAC地址 |
| authtype | 固定为“web”，表示web认证 |
| gw_address | 与重定向参数保持一致 |
| gw_port | 与重定向参数保持一致 |
| gw_id | 与重定向参数保持一致 |
| url | 与重定向参数保持一致 |

第五步：客户端向路由器提交用户名和密码

无线终端在认证登录页面填写用户名和密码后点击 登录 按钮，就以GET的方式将user、pwd等参数提交给路由器。

第六步： 路由器向认证服务器提交认证信息

路由器在获取客户端提交的信息之后，确定需要进行认证的设备，然后把所有的参数提交给认证服务器进行认证。

第七步： 认证服务器向路由器返回认证结果

认证服务器根据路由器提交的信息判断用户是否通过认证，并且向路由器返回认证结果。

第八步： 路由器向客户端返回认证结果

路由器根据认证服务器返回来的结果向无线终端返回相应的认证结果，若认证成功，路由器则根据之前获取的参数信息对相应设备的上网数据给予放行。

Via: http://service.tp-link.com.cn/detail_article_3413.html

药不是糖

Posted on 2017-10-27

这两天，阿崽咳嗽了，昨夜更是反复闹腾，半夜醒来好几次。

今晚回到家，阿崽躺在沙发上看电视。过去摸了摸他的额头，所幸没发烧，也没听到咳嗽声。叫唤了他几声，只听见他回应了一下，声音似乎有点哑。

阿崽抬起头问我：爸爸，那个甜甜的你放哪里了，我找了好久都没找到。

我问“是什么甜甜的？”，“就是你早上给我吃的啊。”，阿崽一说我就明白了，是那板药品 —— 施保利通片。转身走到冰箱旁边拿了给他，我就自己径自吃饭去了。

等他看完电视跑来餐桌边，我全然没注意。后来发现他一颗一颗掰开那一板药，塞到嘴巴里，腮帮子一动一动地嚼着。心里一惊：大事不妙。

果然，他以为这个甜甜的药是糖。

赶紧把他嘴里剩下的药片抠了，漱口。冲了一瓶奶，争取降低下副作用。估摸着回忆了下，应该被他吃了3片还是4片。回去抽屉找到了说明书。嗯，还好，说明书写着过量服用没有什么大碍。

后来，我问阿崽，为什么自己掰开吃。他说“上次吃药的时候，妈妈说这是糖糖”。顿时，恍然大悟！

以后，不能因为哄孩子吃药，而骗他说这是糖糖。意外，总是发生在你没办法预料的地方。

育儿的又一个坑，谨记。

教训了他一顿

Posted on 2017-10-23

今天晚上，扇了儿子一巴掌。

他在床上想玩叠好的衣物，我们没应允。他手里拿着霸王龙，用那尖尖的恐龙尾巴，指着我大叫，“我要戳死你。”

我转身进了房间，对着他的脸蛋，狠狠地打了一巴掌。

待他哭闹完了，给他讲道理：“你戳到别人，别人眼睛看不见怎么办？这样子是不对的，知道吗？”，他点点头，说知道了。

然后，陪着他拿了画笔一起涂画，气氛恢复往常，好像刚才的事情没发生。只是他脸上的几个手指印，看着让我心隐隐的作痛。

或许，之前太过宽容，他在外面学到了很多坏习惯。

这几天，我变得比之前更严肃了。他在家总喜欢拍打人，被我罚了站。看电视经常不肯关掉，没少挨我批评。诸如此类的小毛病，让人头疼。

之前，他在外面被人欺负了，我便教他，别人打了你，你就打回去，男子汉，要勇敢一点。结果第二天，他竟把一个小哥哥给欺负了。

教育，始终是不能掉以轻心。

平时多注重讲道理，但倘若突破了底线，还是得毫不犹豫的体罚一下。有些底线，是要坚决去维护的。

PHP AES/CBC/PKCS5Padding encrypt/decrypt

Posted on 2017-10-13 
class MagicCrypt
{
    // 密钥偏移量IV，可自定义
    private $iv = '';

    // AES-key，可自定义
    private $encryptKey = '';

    public function __construct($encryptKey, $iv)
    {
        $this->encryptKey = $encryptKey;
        $this->iv = $iv;
    }

    /**
     * 加密
     * @param string $encryptStr
     * @return string
     */
    public function encrypt($encryptStr)
    {
        $localIV = $this->iv;
        $encryptKey = $this->encryptKey;

        // open module
        $module = mcrypt_module_open(MCRYPT_RIJNDAEL_128, '', MCRYPT_MODE_CBC, $localIV);

        mcrypt_generic_init($module, $encryptKey, $localIV);

        // padding
        $block = mcrypt_get_block_size(MCRYPT_RIJNDAEL_128, MCRYPT_MODE_CBC);
        $pad = $block - (strlen($encryptStr) % $block); //Compute how many characters need to pad
        $encryptStr .= str_repeat(chr($pad), $pad); // After pad, the str length must be equal to block or its integer multiples

        // encrypt
        $encrypted = mcrypt_generic($module, $encryptStr);

        // close
        mcrypt_generic_deinit($module);
        mcrypt_module_close($module);

        return base64_encode($encrypted);
    }

    /**
     * 解密
     * @param string $encryptStr
     * @return bool|string
     */
    public function decrypt($encryptStr)
    {
        $localIV = $this->iv;
        $encryptKey = $this->encryptKey;

        // open module
        $module = mcrypt_module_open(MCRYPT_RIJNDAEL_128, '', MCRYPT_MODE_CBC, $localIV);

        mcrypt_generic_init($module, $encryptKey, $localIV);

        $encryptedData = base64_decode($encryptStr);
        $encryptedData = mdecrypt_generic($module, $encryptedData);

        return $encryptedData;
    }
}

Example:

$privateKey = '1234567890abcdef';
$iv = 'aes-cbc-iv';
$magicCrypt = new MagicCrypt($privateKey, $iv);
$encryptString = $magicCrypt->encrypt($encryptString);
$decryptString = $magicCrypt->decrypt($encryptString);

var_dump($encryptString);
var_dump($decryptString);
